## installation
```shell
npm install -S @jswork/react-ant-pre-select
```

## usage
1. import css
  ```scss
  @import "~@jswork/react-ant-pre-select/dist/style.css";

  // or use sass
  @import "~@jswork/react-ant-pre-select/dist/style.scss";

  // customize your styles:
  $react-ant-pre-select-options: ()
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
